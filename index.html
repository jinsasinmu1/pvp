Cara Setup GitHub Pages

Ikuti langkah-langkah berikut untuk mengaktifkan GitHub Pages pada repository Anda:

Login ke GitHub

Buka github.com dan login ke akun Anda.

Buka Repository pvp

Masuk ke repository jinsasinmu1/pvp.

Pergi ke Settings

Klik tab Settings di bagian atas repository.

Pilih Pages

Pada menu sidebar, pilih Pages.

Konfigurasi Source

Pada bagian Source, pilih:

  • Branch: main atau master
  • Folder: / (root) atau /docs jika menggunakan docs folder

Simpan Konfigurasi

Klik Save dan tunggu beberapa menit hingga GitHub Pages aktif.

Unggah File Website

Pastikan Anda telah mengunggah file website (index.html) ke repository Anda.

Buka GitHub Pages Settings Buka Repository

Tips dan Troubleshooting

Website Tidak Muncul

Jika website tidak muncul setelah setup, tunggu beberapa menit. GitHub membutuhkan waktu untuk deploy halaman.

Error 404

Pastikan file index.html ada di root directory repository atau di folder yang ditentukan pada settings.

Custom Domain

Anda dapat menggunakan domain sendiri dengan menambahkannya pada bagian Custom Domain di settings GitHub Pages.

Dukungan GitHub

Jika mengalami masalah, kunjungi dokumentasi resmi GitHub Pages.
